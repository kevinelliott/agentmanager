# AgentManager CLI/TUI Design Specification

## Design Philosophy

**Visual Identity**: Professional, modern, developer-friendly
**Tone**: Helpful, clear, efficient
**Color Strategy**: Semantic colors with graceful NO_COLOR fallback
**Accessibility**: Color never the only indicator, screen reader friendly

---

## Color Palette

### Primary Colors
```go
// Semantic colors
Success     = lipgloss.Color("#00D787")  // Bright green
Warning     = lipgloss.Color("#FFB86C")  // Warm orange
Error       = lipgloss.Color("#FF5555")  // Clear red
Info        = lipgloss.Color("#8BE9FD")  // Cyan blue
Muted       = lipgloss.Color("#6272A4")  // Dim purple-gray

// UI Elements
Primary     = lipgloss.Color("#BD93F9")  // Purple (brand)
Secondary   = lipgloss.Color("#50FA7B")  // Green (accent)

Background  = lipgloss.Color("#282A36")  // Dark background

```

### NO_COLOR Support
All commands must detect `NO_COLOR` environment variable and fallback to:
- Bold/dim for emphasis
- Symbols: âœ“ â†’ [âœ“], âœ— â†’ [âœ—], â€¢ â†’ *, etc.
- Box-drawing characters remain (they're structural, not decorative)

---

## CLI Design

### Global Flags
```
--json          Output as JSON (machine-readable)
--quiet, -q     Minimal output
--verbose, -v   Detailed output
--no-color      Disable colors
--help, -h      Show help
```

### Exit Codes
```
0   Success
1   General error
2   Command/usage error
3   Network error
4   Permission error
5   Agent not found
```

## Responsive Behavior

### Terminal Width Breakpoints

```
< 80 columns:   Minimal layout
                - Hide side panels
                - Stack elements vertically
                - Truncate long text
                - Show abbreviated shortcuts

80-120 columns: Standard layout
                - Side-by-side panels
                - Full shortcuts
                - Detailed information

> 120 columns:  Wide layout
                - Three-column layout where applicable
                - Extended descriptions
                - More visible items
```

### Height Adaptation

```
< 24 rows:      Compact mode
                - Fewer list items visible
                - Hide activity log
                - Minimal padding

24-40 rows:     Standard mode
                - Normal list size
                - Show all panels

> 40 rows:      Expanded mode
                - More list items
                - Extended activity logs
                - Additional information panels
```

   - Offer retry options

   - Show exact commands to fix
   - Link to documentation

   - Suggest upgrade paths
   - Link to compatibility docs

   - Cache API responses

   - Run updates in background
   - Show progress without blocking UI
   - Queue multiple operations

   - Cancel downloads on quit
   - Clean up temporary files
   - Release file handles
